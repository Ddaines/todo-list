# Todo List

Приложение для управления списком задач на HTML, CSS и JavaScript.

## Описание

Позволяет добавлять задачи, отмечать их выполненными и удалять. Данные сохраняются в localStorage.

## Стек

- **Frontend:** HTML, CSS, JavaScript (Vanilla)

## Как запустить

Открой файл `index.html` в браузере — готово!

## Источник проекта

Проект выполнен на основе туториала из каталога [project-based-learning](https://github.com/practical-tutorials/project-based-learning#javascript).
